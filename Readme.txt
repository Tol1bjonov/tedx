# TEDxAjouTashkent Official Website 🌐

Welcome to the official repository for the [TEDxAjouTashkent](https://tedx.ajou.uz) event website!

This site shares information about the upcoming TEDx event at Ajou University in Tashkent, including details about our speakers, program, venue, and sponsors.

## 🌟 Live Website
👉 [https://tedx.ajou.uz](https://tedx.ajou.uz)

## 📚 About the Project
This website is built to:
- Share event details and updates
- Introduce TEDxAjouTashkent speakers
- Display sponsors in compliance with TEDx rules
- Provide information about TEDx and TED

## 🚀 Tech Stack
- **Frontend:** HTML, CSS, Bootstrap, JavaScript 
- **Domain:** `tedx.ajou.uz`
